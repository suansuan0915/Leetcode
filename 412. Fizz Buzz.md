# [412. Fizz Buzz](https://leetcode.com/problems/fizz-buzz/description/)

Easy

### Solutions

```python
class Solution:
    def fizzBuzz(self, n: int) -> List[str]:
        ans = list()

        for num in range(1, n+1):
            if num % (3*5) == 0:
                ans.append('FizzBuzz')
            elif num % 3 == 0:
                ans.append('Fizz')
            elif num % 5 == 0:
                ans.append('Buzz')
            else:
                ans.append(str(num))

        return ans
```
