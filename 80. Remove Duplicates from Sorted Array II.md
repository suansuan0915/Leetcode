- Better use for loop for clean version.

- If not: \
While loop needs a counter changing for each round.

One pointer loops through the list to compare duplicate, another point at the backward to take note down of the #, while a variable tracks the duplicate#.

### Solution
- Method:
  
  Two pointers:
  - If dup comparison occurs:
    - if dup count less than 3, assign value to p and moves p forward.
    - if more than 3, keep p and just move mover.
  - No dup comparison: assign value to p, then move p and mover. reassign dup count to 1.
  - mover pointer moves EVERY TIME.

  TC: O(n)\
  SC: O(1)
  
```python
class Solution:
    def removeDuplicates(self, nums: List[int]) -> int:
        if not nums:
            return 0

        p = 1
        dup = 1

        for mover in range(1, len(nums)):
            if nums[mover] == nums[mover-1]:
                dup += 1
                if dup <= 2:
                    nums[p] = nums[mover]
                    p += 1
            else:
                nums[p] = nums[mover]
                p += 1
                dup = 1

        return p
```
