### Solutions
- Method 1:\
  Greedy.\
  TC: O(n)\
  SC: O(1)
```python
class Solution:
    def maxProfit(self, prices: List[int]) -> int:
        min_i = 0
        profit = 0

        for max_i in range(1, len(prices)):
            # if prices[min_i] - prices[max_i] > 0:
            #     min_i = max_i
            if prices[max_i] - prices[min_i] > 0:
                profit += (prices[max_i] - prices[min_i])
            min_i = max_i
        
        return profit
```
