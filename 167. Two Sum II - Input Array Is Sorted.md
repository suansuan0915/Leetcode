## 167. Two Sum II - Input Array Is Sorted

:alarm_clock: Feb 8

### Binary Search: Two Pointers (left, right)

**nSum** Question\
-> left, right pointers

#### Code:
```python

class Solution:
    def twoSum(self, numbers: List[int], target: int) -> List[int]:
        ## Method:
        # Binary Search - Two Pointers (left,right)
        # TC: O(N), SC: O(1)
        l, r = 0, len(numbers)-1
        while l <= r:
            if numbers[l] + numbers[r] == target:
                return [l+1, r+1]
            elif numbers[l] + numbers[r] < target:
                l += 1
            else:
                r -= 1
        return
```
