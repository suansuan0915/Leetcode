# [373. Find K Pairs with Smallest Sums](https://leetcode.com/problems/find-k-pairs-with-smallest-sums/description/?envType=study-plan-v2&envId=top-interview-150)

Medium

### Solution

Cannot use two pointers method.
- Edge case: `nums1 = [1,1,2], nums2 = [1,2,3]`

Idea: Heap
- first pair must be 0th elements of both arrays.
- following pairs can be (i, j+1) or (i+1, j).

Note:
- avoid visiting visited pairs.

TC: O(min(k⋅logk,m⋅n⋅log(m⋅n)))
SC: O(min(k, m⋅n))

```python
class Solution:
    def kSmallestPairs(self, nums1: List[int], nums2: List[int], k: int) -> List[List[int]]:
        m, n = len(nums1), len(nums2)
        hp = [[nums1[0]+nums2[0], (0, 0)]]
        visited = set()
        visited.add((0,0))
        ans = list()
        p1, p2 = 0, 0

        while k > 0:
            val, (p1, p2) = heapq.heappop(hp)
            ans.append([nums1[p1], nums2[p2]])
            if p1+1 < m and (p1+1, p2) not in visited:
                heapq.heappush(hp, [nums1[p1+1]+nums2[p2], (p1+1, p2)]) 
                visited.add((p1+1, p2))
            if p2+1 < n and (p1, p2+1) not in visited:
                heapq.heappush(hp, [nums1[p1]+nums2[p2+1], (p1, p2+1)]) 
                visited.add((p1, p2+1))
            k -= 1

        return ans
```
