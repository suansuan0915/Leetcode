# [173. Binary Search Tree Iterator](https://leetcode.com/problems/binary-search-tree-iterator/description/?envType=study-plan-v2&envId=top-interview-150)

Medium

### Solutions

- Method 1:\
  Flatten BST into a sorted array.

  TC: O(n) - constructor, O(1) - next(), hasNext()\
  SC: O(n)

```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class BSTIterator:

    def __init__(self, root: Optional[TreeNode]):
        self.sorted_nodes = self.flatten(root, [])
        self.idx = -1

    def flatten(self, root, path):
        if not root:
            return path

        self.flatten(root.left, path)
        path.append(root)
        self.flatten(root.right, path)
        return path

    def next(self) -> int:
        self.idx += 1
        return self.sorted_nodes[self.idx].val

    def hasNext(self) -> bool:
        return self.idx + 1 < len(self.sorted_nodes)


# Your BSTIterator object will be instantiated and called as such:
# obj = BSTIterator(root)
# param_1 = obj.next()
# param_2 = obj.hasNext()
```


- Method 2:\
  stack.

  Idea:
  - becoz the initial node starts from most smallest node (leftmost of root), so we keep the leftmost path in stack by traversal,
  the next_smallest is `stack.pop()`.
  - whenever next(), we move one node forward, so need to tidy up new relationship at the new node:
    - if no right child: then keep going on.
    - if right child exists: traverse and add to stack.
  - stack: `[....nodes larger than next.....next_to_pop]`, next_to_pop > next_to_pop's right child, so add to stack as candidates of next smallests.

  TC: O(n) - constructor, O(1) - next(1), hasNext(1)\
  SC: O(h)

```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class BSTIterator:

    ### Method 2:
    ## stack
    def __init__(self, root: Optional[TreeNode]):
        self.stack = []
        self._buildStack(root)
        
    def _buildStack(self, root):
        while root:
            self.stack.append(root)
            root = root.left

    def next(self) -> int:
        next_smallest = self.stack.pop()
        
        if next_smallest.right:
            self._buildStack(next_smallest.right)

        return next_smallest.val

    def hasNext(self) -> bool:
        return len(self.stack) > 0


# Your BSTIterator object will be instantiated and called as such:
# obj = BSTIterator(root)
# param_1 = obj.next()
# param_2 = obj.hasNext()
```
