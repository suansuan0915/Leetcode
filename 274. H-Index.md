### Solutions

- Method 1:\
  sorting, then check intersection with y=x.\
  TC: O(nlgn), SC: O(1).
```python
class Solution:
    def hIndex(self, citations: List[int]) -> int:
        citations.sort(reverse=True)

        for i in range(len(citations)):
            if citations[i] < i+1:
                return i

        return len(citations)
```
